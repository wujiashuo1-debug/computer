## 部署

上传到 GitHub Pages 时，仓库根目录应直接包含：

```text
index.html
style.css
app.js
questions.js
manifest.webmanifest
sw.js
.nojekyll
assets/
README.md
```

不要把外层文件夹整体套进仓库。
